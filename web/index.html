RewardEvents autenticação

Olá, seja bem-vindo ao RewardEvents

Para iniciarmos o aplicativo é necessário efetuar a autenticação das contas streamer e bot, a autenticação é necessária para termos acessos á alguns dados importantes para o funcionamento correto do sistema, fique tranquilo, todos os dados de autenticação são salvos localmente e somente você e a twitch tem acesso ao token de autenticação.

Sobre a conta bot, ela é a conta que responderá aos comandos dos usuários e/ou respostas de status do sistema, então pode ser a sua conta streamer mesmo se preferir, porém recomendamos o uso de outra conta para evitar erros e problemas com as requisições com a twitch e manter tudo mais bonito e organizado no seu chat.

Bom, agora que já esclarecemos algumas coisas, clique no botão abaixo para iniciarmos.
